# Budget Statistic

The Budget Statistic feature is used to estimate and plan statistical data needed for analysis and reporting. It allows users to set budget targets for various statistics, using flexible input methods to match planning requirements.

## Step-by-Step Process to Create a Budget Statistic

1. Select the **Accounting** menu, then click the **Budget Statistic** submenu to open the Budget Statistic window.
2. Click the **Insert** button to add new statistical budget data.
3. Select the accounting year in **Period**.
4. Select the statistical data in **Account**.
5. Choose the input type:
   - **Manual**: Enter monthly budget amounts for each month (January to December). The total will be calculated automatically.
   - **Average**: Enter the total budget in **Amount**. The system will divide the total evenly across all months.
   - **Percentage**: Enter the total budget and specify the percentage for each month until the total reaches 100%.
   - **Daily**: Enter the total budget in **Amount**. The system will divide the budget for each month based on the number of days in that month.
6. Click **Save** to store the budget, or **Close** to cancel.
7. Repeat the process for all required statistical data in the Budget Statistic submenu.
