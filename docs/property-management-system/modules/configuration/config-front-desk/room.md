# Room

The Room configuration page allows you to add, update, and delete rooms in your hotel. Managing room data ensures your property inventory is accurate and helps guests find the right accommodation.

## What is a Room?

A Room represents an individual accommodation unit in your hotel, with specific attributes such as type, bed, view, and location. Properly configuring rooms is essential for operations, bookings, and guest satisfaction.

## How to Add a Room

1. In the Tools menu, click the Configuration submenu, select the Front Desk tab, then click the Room button to open the window.
2. Click the Insert button to add a new room. A new entry window will appear.
3. Enter the following data:
   - **Number**: Room number or identifier.
   - **Room Type**: Select the room type.
   - **Bed Type**: Select the bed type.
   - **Room View**: Select the room view.
   - **Building**: Specify the building if applicable.
   - **Floor**: Specify the floor.
   - **Max Adult**: Maximum number of adults allowed.
   - **Room Smoking**: Indicate if the room is smoking or non-smoking.
   - **Description**: Add a description for the room.
   - **Phone Number**: Room phone number.
   - **Start Date**: Date the room becomes available.
   - **Owner**: Specify the owner if applicable.
4. Upload a room image if desired.
5. Click Save to store the new room.

Managing rooms ensures your hotel inventory is up to date and ready for reservations.
