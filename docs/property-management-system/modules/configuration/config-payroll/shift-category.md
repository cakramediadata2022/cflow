# Shift Category

Classify shifts by type for better organization and reporting.

**Purpose:** Group shifts into categories for filtering and analysis.

**Key Fields:**

| Field             | Type     | Description              | Example                |
| ----------------- | -------- | ------------------------ | ---------------------- |
| **Category Name** | Text     | Category identifier      | Regular Shift          |
| **Description**   | Text     | Category explanation     | Standard 8-hour shifts |
| **Color Code**    | Color    | Visual identifier        | Green                  |
| **Is Overtime**   | Checkbox | Marks shifts as overtime | No                     |
| **Status**        | Toggle   | Active or Inactive       | Active                 |

**Common Categories:**

- Regular Shift
- Night Shift
- Weekend Shift
- Holiday Shift
- On-Call Shift
- Overtime Shift

**How to Use:**

1. Navigate to **Work Schedule** → **Configurations** → **Shift Category**
2. Click **Add Category**
3. Enter category name
4. Provide description
5. Select color code
6. Mark as overtime if applicable
7. Activate category
8. Assign shifts to this category in Shift Configuration
