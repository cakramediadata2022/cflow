# Shift Coverage

Define minimum staffing requirements for operational coverage.

**Purpose:** Ensure adequate employee coverage for each shift and location.

**Key Fields:**

| Field              | Type   | Description             | Example          |
| ------------------ | ------ | ----------------------- | ---------------- |
| **Placement**      | Select | Location                | Branch A         |
| **Department**     | Select | Department              | Customer Service |
| **Shift**          | Select | Target shift            | Morning Shift    |
| **Minimum Staff**  | Number | Required employees      | 5                |
| **Optimal Staff**  | Number | Ideal employee count    | 7                |
| **Day Type**       | Select | Weekday/Weekend/Holiday | Weekday          |
| **Effective Date** | Date   | Coverage rule starts    | 2026-01-01       |
| **Status**         | Toggle | Active or Inactive      | Active           |

**How to Use:**

1. Navigate to **Work Schedule** → **Configurations** → **Shift Coverage**
2. Click **Add Coverage Rule**
3. Select placement and department
4. Choose shift
5. Set minimum staff requirement
6. Set optimal staff level
7. Specify day type
8. Set effective date
9. Activate rule
10. System alerts when coverage below minimum during scheduling
